# Redmi K70Pro 手机电池高功耗使用实验数据 #

## 数据采集背景 

本数据为Redmi K70Pro 手机电池高功耗使用实验数据，用于验证智能手机电池耗电机理模型的准确性，采集自小组实际手机测试。 

## 测试环境 

1. 测试设备：手机型号（红米K70Pro）、系统版本（Xiaomi HyperOS 3.0.7.0）、电池健康度（85%-90%） 

2. 监测工具：Ampere中文版（实时采集电流/电压/温度） 

3. 测试前准备：关闭后台无关APP、关闭省电模式、授予工具“忽略电池优化”权限，避免数据干扰 

   ## 数据字段说明

   | 列名          | 单位 | 说明                           |
   | ------------- | ---- | ------------------------------ |
   | time          | s    | 测试时间戳，每5秒采集1条       |
   | current       | mA   | 电池电流（放电为负，充电为正） |
   | voltage       | v    | 电池实时电压                   |
   | temperature   | ℃    | 电池核心温度                   |
   | battery_level | %    | 剩余电量百分比                 |

   ## **测试场景说明**

   1. 测试类型：高功耗放电测试 
   2. 具体操作：测试期间保持屏幕常亮（亮度50%，120Hz），0-20分钟运行[王者荣耀] ，20-40分钟关闭应用并清理后台
   3. 测试时长：40分钟，有效数据条数：268条

   ## 数据采集规则 

   - 采集频率：5秒/条，无间断采集
